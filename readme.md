There is two folder: backned and fronted.

Firstly:

```
# clone repo
git clone https://github.com/Kastuell/test_app_indevlabs.git


```

Then:

```
# Go to back folder and use this commands:

# install deps
yarn

# insert tables in prisma
prisma db push

# run server
yarn start:dev

```

After server part:

```

# Go to front folder and use this commands:

# install deps
yarn

# run client
yarn dev

```
